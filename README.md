Steps to Open the Website

Open Assignment folder in VS code
1. Extract the Assignment Folder
2. Open Visual Studio Code and click 'Open Folder'
3. Locate the Folder and Click on it, open the folder 'Apps'


Importing the database:
4. Open XAMPP control panel
5. Start both Apache and MySQL
6. Click on 'Config' at the role of MySQL, phpMyAdmin will open
7. At the top level, click 'Import'
8. Select the .sql file located within the assignment folder
9.Click 'Import' at below

Starting the website
10. Open the terminal at Visual Studio Code
11. Type "php -S localhost:8000" and press enter
12. Click on the link given
13. You can enter member or admin side by registering accounts for yourselves.
